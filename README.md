## Роль у системі
Повний пайплайн для count-based моделей: реальний текст – токенiзацiя – тренування
NGram – генерацiя – checkpoint. Це мiнi-демо системи Mini-ChatGPT на базовому рiвнi.

## Word vs Char
Генерація цілих слів з WordTokenizer працює швидше та зазвичай видає текст з більшим сенсом. Потребує словника більшого розміру для навчання.

Посимвольна генерація з CharTokenizer працює повільніше та видає менш зрозумілий результат, оскільки може вигадувати нові слова та "застрягати" на найбільш відомих наборах символів. Для навчання потребує словник відносно невеликого розміру.

## Bigram vs Trigram
При тренуванні обидвох моделей на однаковому корпусі перплексія trigram є нижчою, ніж перплексія bigram, однак це стає помітно лише на відносно великих наборах даних. Для корпусів меншого розмір перплексії обох моделей будуть однаковими через fallback trigram та неможливість переваг trigram "проявити" себе.
